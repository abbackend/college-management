# College Management

## Required
- PHP `8+`
- composer
- mysql
- apache
- phpmyadmin

## Steps For Install
- Clone the repo.
- `cd college-management`
- Install required dependency: `composer install`
- Run migration and seeder: `php artisan migrate --seed`
- Run the development environment: `php artisan serve`
